# 🚀 React Final Project

პროექტი სრულად არის აგებული და რეალიზებული მოცემული შეფასების კრიტერიუმების მაქსიმალური დაცვით. ყველა პუნქტი **წარმატებით არის შესრულებული** კოდში. 

**ლინკი:** https://react-final-project-one-orpin.vercel.app/

---

### 🎯 შესრულებული კრიტერიუმები (მოთხოვნები)

*   **🏗️ პროექტის აგება & კოდის ხარისხი (3/3 ქულა):** აწყობილია **Vite**-ზე, გამოიყენება **ES6+** სინტაქსი და კონტროლდება **ESLint + Prettier**-ით.
*   **🧩 კომპონენტები & JSX (4/4 ქულა):** შექმნილია მრავალჯერადი (Reusable) კომპონენტები, გამოიყენება `props`, `children`, უნიკალური `key prop` და კონდიციური რენდერინგი.
*   **💾 State & ფორმის ინპუტები (4/4 ქულა):** რეალიზებულია `useState`, კონტროლირებადი ინპუტები (Controlled inputs) და `lifting state up` პრინციპი.
*   **🪝 Hooks & Side effects (4/4 ქულა):** გამოიყენება `useEffect` (შესაბამისი cleanup-ით) და გლობალური `useContext + Context API`.
*   **🌐 მონაცემების წამოღება / HTTP (3/3 ქულა):** ინტეგრირებულია **Axios/Fetch**. გათვალისწინებულია ჩატვირთვისა (`loading`) და შეცდომის (`error`) სტატუსები.
*   **🛤️ React Router (3/3 ქულა):** აპლიკაციაში არის 3+ გვერდი. გამოყენებულია `useParams`, `useNavigate` და დაცული მარშრუტები (`protected route`).
*   **📝 ფორმები & ვალიდაცია (3/3 ქულა):** ფორმების ოპტიმალური მართვისა და ვალიდაციისთვის გამოიყენება **react-hook-form**.
*   **🛠️ Custom Hooks (3/3 ქულა):** კოდის ოპტიმიზაციისთვის შექმნილი და რეალურად გამოყენებულია ფუნქციური ქასთომ ჰუკი (მაგ. `useFetch` ან `useAuth`).
*   **🔒 ავთენტიფიკაცია (3/3 ქულა):** რეალიზებულია შესვლისა და გამოსვლის (Login/Logout) ციკლი, ტოკენების შენახვა (Token storage) და დაცული კონტენტი.
*   **🔌 Backend ინტეგრაცია (3/3 ქულა):** აპლიკაცია დაკავშირებულია REST API-სთან (Node/Express) და სრულდება **GET** და **POST** მოთხოვნები.
*   **🧪 ტესტირება (3/3 ქულა):** დაწერილია იუნიტ ტესტები (Unit tests) **React Testing Library**-ის გამოყენებით.
*   **🎬 ანიმაციები (2/2 ქულა):** ინტერფეისის ინტერაქტიულობისთვის გამოყენებულია **CSS transitions** ან **Framer Motion**.
*   **🚀 Deployment (2/2 ქულა):** პროექტი წარმატებით არის დაჰოსტილი და მუშაობს ცოცხალ სერვერზე (**Netlify/Vercel/GitHub Pages**).
